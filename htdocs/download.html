For installation guidence, read the getting started guide in the AssaultCube documentation.
Linux users should take special note that they will need these additional libraries installed:

  • SDL 1.2
  • SDL_image 1.2
  • OpenAL Soft
  • SDL_ttf 2.0

If you're on a Debian or Ubuntu based system, you can run this command in a terminal to install these libraries:
sudo apt-get install libsdl1.2 libsdl-image1.2 libopenal1 libsdl-ttf2.0-0
